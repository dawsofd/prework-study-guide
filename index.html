Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
    • text elements
    • image elements
    • header elements
    • list elements

CSS

  • CSS specifies the presentation and styling of a document
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A period preceeding a defined element designates a class selector. The styling defined will apply to any element with that class.

Git

  • Git flow:
    • git checkout -b feature/NAMEOFBRANCH
    • git status
    • git add -A
    • git commit -m "ADD COMMENT"
    • git push origin feature/NAMEOFBRANCH
    • create pull request
    • merge

Javascript
